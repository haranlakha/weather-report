# Weather Report REST Program

Weather Report REST program using [OpenWeatherAPI](https://openweathermap.org/api) data.

## Setup

You will need to create an account with OpenWeatherAPI and get an API key. This is stored in the config.properties file.

## Installation

```
git clone https://github.com/haranlakha/weather-report-rest.git
```

## Compile code

Build the application in your favourite IDE e.g. [IntelliJ IDEA](https://www.jetbrains.com/idea/).

## Run code

Run the application in your IDE.
